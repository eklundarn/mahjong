# 🀄 Mahjong Club Manager

A web-based system for managing a mahjong club — game registration, player statistics, rankings, and tournament management. Built with PHP and MySQL, designed for mobile-first use during game sessions.

Originally developed for [Varbergs Mahjongsällskap](https://varbergmahjong.se) (Varberg Mahjong Society, Sweden).

## Features

- **Mobile game registration** — Register MCR mahjong games hand by hand on your phone, with automatic score calculation, self-drawn/discard tracking, and penalty support
- **Player management** — Member database with VMS/EMA numbers, roles, and permissions
- **Statistics & rankings** — Yearly rankings, head-to-head stats, player profiles, game history
- **Tournament system** — Full tournament management with seating, live scoring, ceremony system, and printable player cards
- **Offline support** — Games are saved locally and queued for sync when connection returns
- **Bilingual** — Swedish and English throughout (language detection + manual switch)
- **Responsive** — Mobile-first design, works on phones, tablets, and desktops

## Requirements

- PHP 7.4 or higher
- MySQL 5.7+ or MariaDB 10.3+
- PDO MySQL extension
- Web server (Apache, Nginx, or similar)
- SMTP access for email features (optional)

## Installation

### 1. Download and upload

Download or clone this repository and upload the contents of the `www/` folder to your web server's document root.

### 2. Run the installer

Open `https://your-domain.com/install.php` in your browser.

The installer will:
- Check system requirements
- Collect your database credentials, club name, and admin account details
- Create all required database tables (14 tables)
- Generate `config.local.php` with your settings
- Create your admin account

### 3. Delete install.php

**Important!** After installation, delete `install.php` from your server for security.

### 4. Log in

Go to `https://your-domain.com/login.php` and log in with the admin account you created.

## Configuration

All configuration is in `config.local.php` (generated by the installer). Key settings:

| Setting | Description |
|---------|-------------|
| `DB_HOST`, `DB_NAME`, `DB_USER`, `DB_PASS` | Database connection |
| `SITE_URL` | Full URL of your site (no trailing slash) |
| `SITE_TITLE` / `SITE_TITLE_EN` | Club name in local language / English |
| `SMTP_*` | Email settings for notifications and password resets |
| `REQUIRE_PLAYER_CONFIRMATION` | If `true`, all 4 players must confirm before admin approval |
| `SESSION_TIMEOUT` | Inactivity timeout in seconds (default: 1800 = 30 min) |
| `REGISTRATION_SESSION_TIMEOUT` | Timeout for game registration pages (default: 14400 = 4 hours) |

## User roles

| Role | Access |
|------|--------|
| `readaccess` | View statistics and game history |
| `clubplayer` | Register games, view own statistics |
| `admin` | All of the above + manage users, approve games, tournament admin |
| `mainadmin` | All of the above + system configuration |

## Adding a tournament

1. Create a new folder under `tournament/` (e.g., `tournament/mytournament/`)
2. Create the tournament database tables using the prefix pattern (e.g., `mytour_tournament`, `mytour_registrations`, `mytour_rounds`, `mytour_tables`, `mytour_results`, etc.)
3. Add the tournament to `tournament/index.php`
4. Add a permission type in the `permission_types` table
5. Grant the permission to tournament admins via the user management page

## Scoring system

The system uses MCR (Mahjong Competition Rules) scoring:
- **Minipoints (MP)**: Raw point score from each game
- **Table points (TP)**: 4/2/1/0 based on ranking at the table (with equal sharing for ties)
- Rankings are sorted by total TP, with total MP as tiebreak
